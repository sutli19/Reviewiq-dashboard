# ReviewIQ – AI-Powered Ecommerce Review Intelligence Dashboard

ReviewIQ is an AI-powered Ecommerce Review Intelligence Dashboard designed to help ecommerce brands monitor, analyze, and act on customer feedback collected from multiple online platforms.

Built as a frontend prototype, the platform transforms large volumes of customer reviews into actionable insights, helping businesses identify trends, improve customer satisfaction, and prioritize issues efficiently.

---

## Live Demo

https://reviewiq-dashboard-nine.vercel.app/

---

## Problem Statement

Ecommerce brands receive thousands of customer reviews every month across platforms such as:

* Amazon
* Flipkart
* Google Reviews
* Instagram
* Brand Websites

Manually analyzing reviews becomes increasingly difficult as brands scale.

ReviewIQ addresses this challenge by providing a centralized dashboard that helps brands:

* Monitor overall sentiment
* Detect recurring issues
* Analyze customer feedback patterns
* Generate actionable insights
* Track issue resolution workflows

## Installation

Clone the repository

```bash
git clone https://github.com/sutli19/Reviewiq-dashboard.git
```

Navigate into the project

```bash
cd Reviewiq-dashboard
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```
